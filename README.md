# SF Muni Tracker

This web app shows the locations of SF MUNI vehicles.

## Setup Grunt

```
npm install
```

## Install libraries

```
bower install
```

## Development

Run `grunt serve` for preview.

## Create CSS

```
lessc less/main.less app/styles/main.css
```

## Deployment

Folder "app" is ready for deployment.
